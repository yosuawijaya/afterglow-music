# Social Media Icons

Upload your social media icon files here:

- instagram.svg
- twitter.svg
- facebook.svg
- spotify.svg

Recommended:
- Format: SVG (scalable)
- Size: 32x32px or larger
- Color: White/Light (will be filtered by CSS)

## Verified Badge

- **verified.svg** - Centang biru yang muncul di sebelah "Afterglow Music" di halaman news

### Cara Mengganti Verified Badge:
1. Buat atau download SVG verified badge yang Anda inginkan
2. Replace file `verified.svg` dengan design Anda
3. Pastikan ukuran SVG sekitar 24x24px untuk hasil terbaik
4. Badge akan otomatis muncul di semua halaman news untuk author "Afterglow Music"
